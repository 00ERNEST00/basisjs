Каждый basis.ui.Node может иметь Selection. Selection - это датасет, в котором хранятся выбранные (selected) на текущий момент дочерние узлы компонента.

Selection может быть одинарным, т.е содержать только один узел, или множественным, т.е содержать сразу несколько узлов. Для добавления Selection к компоненту используется свойство selection. По умолчанию создается одинарный Selection.

Для добавления узла в Selection используется функция узла select. При этом если Selection одинарный, то предыдущий выбранный элемент будет удален из Selection'а.
