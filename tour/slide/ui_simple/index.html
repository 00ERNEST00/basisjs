Благодаря механизму авто-расширения, мы можем упростить код предыдущего слайда. Экземпляры basis.template.html.Template имеют метод __extend__, в случае если задать экземпляр класса basis.template.html.Template, то все произойдет без изменений. В противном случае будет создан новый шаблон (экземпляр basis.template.html.Template), у которого источником будет заданое значение.

Так же при создании экземпляра basis.ui.Node можно задать свойство container, это ссылка на элемент, в который должен быть вставлен корневой элемент шаблона.
