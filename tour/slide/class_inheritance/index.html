Для создания классов используется функция basis.Create. Класс, от которого наследуется создаваемый класс (родительский класс), передается первым параметром.

Для вызова метода супер-метода (метода родителя) используется конструкция вида:
SuperClass.prototype.method.call(this, args...), где SuperClass - родительский класс, а args... перечень параметров.

Так же у каждого класса есть метод subclass, который пораждает новый класс, унаследованный от класса, чей метод был вызван. Следующие две записи эквивалентны:

  var MyClass = basis.Class(SomeClass,  { .. });
  var MyClass = SomeClass.subclass({ .. });
