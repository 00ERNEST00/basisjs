На слайде представлен пример организации наследования в basisjs.

Для создания классов используется функция basis.Create. Наследуемый класс передается первым параметром.

Для вызова метода супер-метода используется конструкция вида:
SuperClass.prototype.method.call(this), где SuperClass - родительский класс.

При создании экземпляра класса вызывается фукнция init.
