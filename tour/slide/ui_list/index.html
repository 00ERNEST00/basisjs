На слайде преставлен пример создание простых компонентов (узлов).

Все базовые визуальные компоненты в basisjs являются экземплярами класса basis.ui.Node или его производных. Каждый экземпляр basis.ui.Node может являться контейнером себе подобных компонентов, которые также будут экземплярами basis.ui.Node или его производных.

Для задания списка дочерних узлов при создании узла используется свойство конфига childNodes. Также есть возможность добавлять узлы, используя функции appendChild и insertBefore. Для удаления узлов используется функция removeChild.
