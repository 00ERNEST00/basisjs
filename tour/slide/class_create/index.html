На слайде представлен пример создания простого класса.

Классы в basisjs создаются с помощью функции basis.Class.create. Функция принимает два параметра: первый параметр - это класс, от которого создаваемый класс будет наследоваться, второй - прототип создаваемого класса. В качестве первого параметра можно использовать null, если создаваемый класс не наследуется от любого другого.

Вместо фукнции basis.Class.create можно использовать ее короткую запись basis.Class.
Например: var MyClass = basis.Class(null, {});
