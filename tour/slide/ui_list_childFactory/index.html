Для создания экземпляров дочерних узлов используется внутренняя функция узла childFactory. По умолчанию childFactory === function(config){ return new this.childClass(config); }

В том случае, когда класс дочернего узла нельзя определить изначально или дочерние узлы могут быть разных классов, функцию childClass можно переопределить, указав в ней логику определения класса дочернего элемента по переданному конфигу.
