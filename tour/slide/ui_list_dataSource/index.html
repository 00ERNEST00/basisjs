Одной из основных возможностей basisjs является использование датасетов (наборов данных) для гененирования узлов. При этом функции appendChild/insertBefore/setChildNodes теряют силу и отходят на второй план, так как всю работу по добавления и удалению узлов берет на себя датасет.

Для связки компонента с датасетом используется свойство dataSource при создании компонента. После чего при добавлении/удалении данных в датасете, такие же изменения буду происходить и в компонете в виде добавленных/удаленных узлов. При этом каждому узлу будет делегирован свой объект basis.data.DataObject из датасета.
