На слайде представлен пример создания простого списка.

Все базовые визуальные компоненты в basisjs являются экземплярами класса basis.ui.Node или его производных. Каждый экземпляр basis.ui.Node может являться контейнером себе подобных компонентов, которые также будут экземплярами basis.ui.Node или его производных.

Чтобы задать класс для дочерних узлов используется свойство childClass, куда передается конфиг нового класса. Для задание списка дочерних узлов используется свойство childNodes.

В данном примере список и его дочерние узлы имеют собственные темплейты, которые храняется в отдельных файлах.
